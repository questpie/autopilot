# @questpie/autopilot-agents

Agent system prompt templates and prompt builder for QUESTPIE Autopilot.

Builds system prompts for agent sessions by combining role definitions, company context, and available tool descriptions into a structured prompt. Used by the orchestrator to configure agent sessions before spawning them.

> **Not meant to be used standalone.** This package is consumed by [`@questpie/autopilot-orchestrator`](https://www.npmjs.com/package/@questpie/autopilot-orchestrator).

## Dependencies

- `@questpie/autopilot-spec` — shared schemas and types

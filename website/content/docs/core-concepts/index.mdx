# Core Concepts

QUESTPIE Autopilot is built around four primitives:

| Concept | Description |
|---------|-------------|
| **Workspace** | A repo root — auto-detected from your current directory |
| **Project** | A specific initiative with its own task graph and state |
| **Session** | A run history record tracking execution results |
| **Task** | An atomic unit of work with states, dependencies, and priority |

These four concepts compose into a dependency-driven execution engine that answers: *what is ready now, what is blocked, and what should run next?*
